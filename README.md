# SpaceShooter
## Что это за проект?
Игра где вы играете за пришельца-уборщика, вам предстоит собирать космический мусор и уклонятся от астероидов.
## Управление:
* WASD
* Enter - выбор
* Esp - меню/пауза
* L Shift - выстрел
* Tab - покупка потронов
